tutorial

Use the sample files available in download

1. load grid: [grid] (press button with this label)
1.1. add contours: edit [contours], ex: 200 1000
1.2. add contour labels: [clabel]
1.3. overlay coastline: [SpectrHA]-->[overlay]-->[coastline]
2. load stations: [station]
3. select station: [select] or [sta#]
(ex: 3; if stations were matrix instead of vector (like when using roms_his, sta# showld be ex: 3 4
4. select variable in popupmenu with first label [ZETA]
5. select vertical level in popupmenu with first label [1]
5.1. or choose depth: select checkbox below sta# and edit [z] (depths below 0 are positive)
6. execute Fourier Spectral Analysis: [FSA]
7. execute Least Squares Fit: [LSF]
8. select radiobutton [ellipse]
9. execute Least Squares Fit: [LSF]
10. click in the output area (to select one ellipse); click again twice to return to original state
11. load ellipse file: [struc]
12. press [hold]
13. execute [LSF]
14. load data file (example_1Dserie.mat): [data]
15. execute [LSF]
16. press [data] to see current data
17. execute [t_tide]
17.1. check t_tide executed command: [SpectrHA]-->[t_tide str]
18. select radiobutton [station] to use data from stations file again
19. use default t_tide, without phase correction: [SpectrHA]-->[phases (t_tide)]-->[default] and run [t_tide]
20. release spectrum: [SpectrHA]-->[release]-->[spectrum]
21. see s-levels: [SpectrHA]-->[release]-->[s-levels (...)]
22. configure LSF: [SpectrHA]-->[config]-->[LSF]
...
