Installation:
put tar.gz file in MATLAB/toolbox (for instance);
cd MATLAB/toolbox;
tar xvfz SpectrHA.tar.gz
add created directories (SpectrHA; SpectrHA/apps; SpectrHA/fprivate; SpectrHA/config) to matlab path
and finally run spectrha
