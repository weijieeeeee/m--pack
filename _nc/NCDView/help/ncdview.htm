NCDView is a Matlab user interface to plot NetCDF files

Features:
- Any 4-D NetCDF variable can be plotted
- supports 1-D, 2-D, and 3-D plots
- several NetCDF files can be used and their variables conjugated
- supports velocity fields (arrows)

Requirements:
- Matlab (created in version 6, but should work in other recent versions)
- NetCDF-Matlab interface
