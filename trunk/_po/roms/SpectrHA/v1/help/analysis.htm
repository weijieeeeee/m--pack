analysis

# Fourier spectral analysis
- press button [FSA]

# Harmonic least squares fit - press button [LSF]
- when radio button [ellipse] is selected, LSF used 2D data and draws ellipses for the desired frequencies
- frequencies are choosed in file S_config_LSF.m
- Rayleigh criteria is not applied
- if performing tidal analysis, the names of tidal constituents can be used if T_TIDE is installed

# T_TIDE - press button [t_tide]
- similar to LSF, but specific to tidal analysis
- T_TIDE toolbox must be installed
- if there is no start time associated with serie being analyses, T_TIDE calculated phases at central time
to avoid this, select [SpectrHA] >> [phases (t_tide)] >> [t=0]
- configuration is done in file S_config_T_TIDE.m
- a part of T_TIDE package is t_predic (tidal prediction); it can be done after a LSF or T_TIDE analysis
cos it uses the output of this routines; start and end time, as well as dt is choosed in the boxes near [predic] button
