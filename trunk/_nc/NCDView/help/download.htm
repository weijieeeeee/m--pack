NCDView

installation:

  1. copy file to $MATLAB/toolbox
  2. tar xvfz NCDView.tar.gz (in windows, you may use winrar)
    this will create the dir NCDView dir with subdirs config, datasets, files, help, tools and toolsnc
  3. add NCDView, tools and toolsnc to path.
    Actually you don't need to add tools and toolsnc to path, they are added to path when ncdv starts (as well as the others subdirs), but specially toolsnc has some functions useful to work with netcdf files, some are not even used by NCDView; check these dirs
  4. that's all, now just run >> ncdv at matlab prompt
  5. be sure you have NetCDF/Matlab interface
